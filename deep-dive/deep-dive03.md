# Deep Dive 3. 7장 연산자

### _연산자_

- `+` 는 산술 연산자 이면서 문자열 연결/결합 연산자
- 나눗셈은 평가 결과 값이 실수임 (배정수)
- 증/감 연산자는 오해의 소지 있음
- 비교 연산자
  - == vs. ===
- NaN 끼리는 항상 같지 않음
  - 무엇이 들어올지 모름
  - Object.is 메서드 이용해서 체크
- 삼항 조건 연산자
  - 조건식에 NaN이 오면 false로 처리
- 단축 평가
- 드 모르간의 법칙
- 지수 연산자는 이항 연산자 중 우선순위 가장 높음
  - 오해 소지 있음 => 소괄호 이용!
- 옵셔널 체이닝 연산자
- null 병합 연산자
